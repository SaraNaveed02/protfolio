=====================================
COMMON CUSTOMIZATIONS
=====================================

Change Colors:
  Edit src/app/globals.css
  
  :root {
    --primary: #a855f7;     /* Change this color */
    --accent: #ec4899;      /* Or this one */
  }

Change Animation Speed:
  Edit src/utils/animations.ts
  
  Change the duration values (in seconds)
  e.g., duration: 0.6 to duration: 1.0

Change Fonts:
  Edit src/app/layout.tsx
  
  Import different font from next/font/google

Change Background:
  Edit src/app/globals.css
  
  body {
    background: your-color-here;
  }
